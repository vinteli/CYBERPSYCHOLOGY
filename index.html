Apa Itu Adiksi Game?

Adiksi game terjadi ketika seseorang bermain game secara berlebihan hingga mengganggu aktivitas harian, tidur, sekolah atau relasi sosial. Kondisi di mana seseorang tidak bisa mengendalikan kebiasaan bermain game, sehingga berdampak negatif pada kehidupan sehari-hari.

Self-Check: Apakah Kamu Kecanduan Game?

1. Saya sering merasa sulit berhenti bermain meski sudah berjam-jam.

2. Saya sering mengabaikan tugas/kegiatan karena game.

3. Saya merasa cemas atau marah saat tidak bisa bermain.

4. Saya sering tidur larut karena main game.

5. Game membuat saya menjauh dari keluarga/teman.



Tips Game-Smart Habits

  • Atur jadwal bermain maksimal 1-2 jam per hari
  • Gunakan alarm untuk mengingatkan waktu istirahat
  • Jangan bermain sebelum menyelesaikan tugas utama
  • Aktif bersosialisasi di dunia nyata
